2.패키지
- Java 프로그래밍 언어를 활용해서 어떤 프로그램을 만들고자 할 때, 정말 다양한 프로젝트 파일들이 필요합니다. 동일한 형제 요소로 생성해놓는 경우, 버그 발생 또는 직관적으로 프로젝트 파일을 찾기가 불편할 수 있음

- 패키지 개념 도입 -> 유사한 성격을 가지고 있는 요소들을 한데 모아놓을 수 있도록 일종의 폴더를 생성할 수 있도록 해주었는데 -> 패키지!!!

- 패키지가 생성되면, 해당 패키지 안에서 코드를 작성하고 있다라는 것을 선언!!!
- 패키지 안에 프로젝트 파일생성 시, 자동으로 상단 package 명령어!!!

- 패키지 안에 또다른 패키지를 만들어서 사용!!!
- 첫번째 패키지명.두번째 패키지명
